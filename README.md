# Mortgage Calculator

## Purpose
This project is designed to help users calculate their mortgage payments based on the principal, interest rate, and loan term. It aims to provide a user-friendly interface for understanding mortgage calculations.

## Tech Stack
- **Language**: Java

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/arti211/MortgageCalculator.git
   ```
2. Navigate into the project directory:
   ```bash
   cd MortgageCalculator
   ```
3. Compile the Java files:
   ```bash
   javac Main.java
   ```
4. Run the application:
   ```bash
   java Main
   ```
5. Follow the on-screen instructions to input loan details.
